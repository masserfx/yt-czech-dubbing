Autentizace

Vsechny API requesty vyzaduji API klic v hlavicce.

Authorization: Bearer vd_live_xxxxxxxxxxxxxxxxxxxx

Preklad videa

POST /v1/translate Zahajit preklad videa

Request Body

ParametrTypPopis
source_url requiredstringURL videa (YouTube, Vimeo, primo MP4)
target_language requiredstringCilovy jazyk: cs, sk, pl, hu
voice_modestringauto (detekce mluvcich) | single (jeden hlas) | custom
voice_idstringID vlastniho hlasu (pouze pro voice_mode: custom)
qualitystringstandard | premium (vyssi kvalita prekladu)
webhook_urlstringURL pro notifikaci po dokonceni
glossary_idstringID firemniho slovniku pro konzistentni terminologii

Priklad requestu

curl -X POST https://api.voicedub.ai/v1/translate \
  -H "Authorization: Bearer vd_live_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "source_url": "https://youtube.com/watch?v=example",
    "target_language": "cs",
    "voice_mode": "auto",
    "quality": "standard",
    "webhook_url": "https://yourapp.com/webhooks/voicedub"
  }'

Response (202 Accepted)

{
  "job_id": "job_a1b2c3d4e5f6",
  "status": "processing",
  "estimated_duration_sec": 120,
  "created_at": "2026-04-16T10:00:00Z"
}
POST /v1/translate/text Prelozit text s TTS syntezou

Request Body

ParametrTypPopis
text requiredstringText k prekladu (max 5000 znaku)
source_languagestringZdrojovy jazyk (auto-detect pokud nevyplneno)
target_language requiredstringCilovy jazyk
voice_rolestringM (muzsky) | F (zensky) | C (detsky) | N (narrator)
output_formatstringmp3 | wav | ogg (default: mp3)

Response (200 OK)

{
  "translated_text": "Prelozeny text v cestine...",
  "audio_url": "https://cdn.voicedub.ai/audio/abc123.mp3",
  "duration_ms": 3200,
  "detected_language": "en",
  "voice_used": "cs-CZ-AntoninNeural"
}

Webhooky

Po dokonceni prekladu posleme POST na vas webhook_url:

{
  "event": "job.completed",
  "job_id": "job_a1b2c3d4e5f6",
  "status": "completed",
  "result": {
    "audio_url": "https://cdn.voicedub.ai/results/job_a1b2.mp3",
    "subtitle_url": "https://cdn.voicedub.ai/results/job_a1b2.srt"
  },
  "timestamp": "2026-04-16T10:02:05Z"
}

Webhook obsahuje HMAC-SHA256 podpis v hlavicce X-VoiceDub-Signature pro overeni autenticity.

Rate limity

Free: 10 req/min, 100 min prekladu/mesic
Pro: 60 req/min, 500 min/mesic
Business: 120 req/min, 2000 min/mesic
Enterprise: Custom limity dle SLA

Rate limit info je v response hlavickach: X-RateLimit-Remaining, X-RateLimit-Reset
